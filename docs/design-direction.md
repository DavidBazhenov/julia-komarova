# Design Direction

## Core Palette

- Background: `#FFFCEB`
- Primary accent: `#325118`

## Design Intent

Сайт должен ощущаться как галерейное пространство, а не как шаблонный e-commerce или generic portfolio. Визуально направление строится на сочетании:

- теплого музейного фона
- глубокого природного зеленого акцента
- крупных художественных изображений
- спокойной типографики с editorial-характером
- большого количества воздуха и ритма

## Выводы по референсам

Из присланных референсов стоит взять не копирование блоков, а набор принципов:

- hero должен быть атмосферным и почти выставочным
- карточки работ должны выглядеть как экспозиция, а не как маркетплейс
- секции должны чередовать светлые и насыщенные зеленые плоскости
- типографика должна быть элегантной, с заметным контрастом между display и body
- интерфейс должен опираться на фотографию и живопись, а не на декоративный UI

## Visual System

### Цвет

- Основной фон страниц: `#FFFCEB`
- Темные акцентные блоки: `#325118`
- Текст на светлом фоне: глубокий теплый почти-черный, не чистый `#000000`
- Текст на зеленом фоне: светлый теплый белый
- Линии и бордеры: очень мягкие, с низким контрастом

### Контраст секций

- `Hero`: крупное полноэкранное изображение с деликатным overlay
- `Content sections`: светлый фон `#FFFCEB`
- `Statement / Highlight sections`: зеленый блок `#325118`
- `Gallery`: светлый фон, чтобы сами работы были в центре

### Типографика

- Для заголовков нужен serif или high-contrast editorial style
- Для основного текста и навигации нужен нейтральный readable sans-serif
- Заголовки крупные, с уверенным вертикальным ритмом
- Текст не должен быть плотным, нужен воздух

## UI Principles

- Максимум внимания картинам, минимум шумного интерфейса
- Навигация легкая и почти невидимая
- Карточки работ должны иметь clean framing и ровную сетку
- Hover effects деликатные: scale, opacity, subtle shift, но без визуального шума
- Кнопки спокойные, с gallery-like характером, не как SaaS controls

## Page Mood

### Home

- Атмосферный fullscreen hero
- Далее спокойные секции с большим количеством negative space
- Featured works как curated selection, а не просто grid

### Gallery

- Строгая сетка
- Фильтры компактные и аккуратные
- Сильный акцент на самих изображениях

### Artwork Page

- Главный фокус на крупном изображении
- Метаданные вторичны визуально, но легко читаются
- CTA на связь должен быть сдержанным

### Admin

- Без декоративности
- Максимум читаемости и скорости работы
- Контраст и структура важнее брендинга

## Design Constraints for Implementation

- Не использовать `Tailwind`
- Не использовать UI frameworks
- Не делать фиолетовые и холодно-серые палитры
- Не перегружать анимацией
- Не строить интерфейс как магазин с ценовым давлением

## MVP Design Deliverables

- Палитра и tokens
- Типографическая шкала
- Сетка и container rules
- Section templates для landing
- Artwork card
- Gallery grid
- Artwork detail layout
- Admin UI baseline
