# Project Docs

Эта папка фиксирует рабочие правила и delivery-план проекта сайта Julia Komarova.

## Состав

- `backlog.md` - подробный backlog следующего этапа, acceptance criteria, зависимости и ownership
- `sprints.md` - спринты, задачи, зависимости, deliverables
- `agents.md` - распределение задач между субагентами и правила взаимодействия
- `clean-code.md` - обязательные инженерные правила проекта
- `i18n.md` - стратегия мультиязычности `en/ru`, правила контента и миграции
- `design-direction.md` - визуальное направление, палитра и UI-принципы
- `release-checklist.md` - предрелизная проверка окружения, функций и выката
- `backup-checklist.md` - backup/restore правила для MongoDB и `storage`
- `smoke-test.md` - сценарий ручной приемки
